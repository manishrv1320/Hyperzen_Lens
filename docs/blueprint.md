# **App Name**: Verdant Vision

## Core Features:

- Image Upload: Allow users to upload images of plant leaves.
- Disease Detection: Use a pre-trained machine learning model to detect plant diseases from uploaded images.
- Disease Information: Display information about the detected disease, including symptoms and treatment options.
- Prevention Tips: Suggest preventive measures to avoid future disease outbreaks, making use of an AI tool to adjust the recommendations based on the detected plant and disease.
- User-Friendly Interface: Create a simple and intuitive interface for easy navigation and image uploading.

## Style Guidelines:

- Primary color: Deep green (#386641) to evoke nature and health.
- Background color: Very light green (#F0F5EF), providing a clean and natural feel.
- Accent color: Earthy brown (#A68A64), drawing focus to key UI elements like the upload button.
- Headline font: 'Alegreya', a serif with an elegant, intellectual feel.
- Body font: 'Inter', a grotesque sans-serif with a modern feel.
- Use nature-inspired icons that are simple and easily recognizable.
- A clean and organized layout with a focus on visual clarity and ease of use.
